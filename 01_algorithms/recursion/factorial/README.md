# Recursive Factorial in C

Computes the factorial of a non-negative integer using recursion.

**Complexity:** Time O(n), Space O(n)

**Compile & Run:**
```bash
gcc main.c factorial.c -o factorial
./factorial
```

**Structure:**
- `main.c`: Entry point with user input/output
- `factorial.c`: Recursive implementation
- `factorial.h`: Function declaration
- `docs/`: Multi-level documentation
