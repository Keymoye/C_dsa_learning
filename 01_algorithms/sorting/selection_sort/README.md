# Selection Sort in C

A simple C implementation of the selection sort algorithm for ordering an array of integers.

- **Complexity**
  - Time: O(n²)
  - Space: O(1)

- **How to compile and run**
  - `gcc main.c selection_sort.c -o selection_sort`
  - `./selection_sort`

- **Folder structure**
  - `main.c` – program entry point and sample array
  - `selection_sort.c` – selection sort implementation
  - `selection_sort.h` – function declaration
  - `docs/` – multi-level learning module
