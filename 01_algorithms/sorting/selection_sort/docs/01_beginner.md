# Selection Sort for Beginners

Selection sort is a way to put numbers in order by repeatedly picking the smallest item and moving it to the front.

## What problem it solves

It sorts a list of values from smallest to largest.

## Real-world analogy

Imagine arranging books on a shelf by height. You scan the shelf, find the shortest book, move it to the first position, then repeat for the remaining books.

## High-level intuition

- Start at the beginning of the list.
- Find the smallest item among the unsorted items.
- Swap it with the item at the current position.
- Move one step forward and repeat.

## Why this algorithm exists

Selection sort is easy to understand and useful for learning how sorting works. It shows a clear process of choosing the next smallest item and building a sorted portion of the list.
